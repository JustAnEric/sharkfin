Integrations
Configure additional Integrations to run with Roblox.
Activity Tracking Permissions
Configure what you want sharkfin (and it's mods) to track from your Roblox Activity.
Some of these features are required by some sharkfin features or sharkfin mods. Disabling some of these options may cause them to not work properly.
Experiences
Allow sharkfin to track what game you are currently playing in Roblox.
Experience History
Allow sharkfin to track what games you've recently played.
Server Location
Allow sharkfin to track the location of the server you are playing on.
Chat Message and Player Join Tracking
Allow sharkfin to track Sent and Received Messages and players joining and leaving from the game you are currently playing on.
Discord Rich Presence
Display the current Roblox Game you are playing in Discord.
This feature requires Experiences tracking to be enabled to run properly.
Discord RPC
Enables this feature and displays the current Roblox Game you are playing.
Allow RPC Joining
Allow anyone to join your game through your Discord Profile
Display Roblox Account
Display the current Roblox Account you are playing as.
Program Autolaunch
Run other programs to launch with Roblox automatically.
Feature not Available

Respect User FastFlags
Enabling this feature makes sure that sharkfin does not override your current existing FastFlags settings.
This may cause some following FastFlags below to not apply.
Verify FastFlags
Enabling this feature allows sharkfin to verify the FastFlag that you are adding to your list.
This will attempt to highlight any FastFlags that does not exist in your config.

Disable Toast Notifications

Disable In-Game Purchases
Disables In-Game Purchases. Good option for heavy spenders!
Disable Beta Translation System Message
Disables the "Roblox automatically translates supported languages in chat." system message on join.

Performance Flags
Increase your Roblox FPS by ~35%! Highly recommended for low-end systems that struggles with Rendering like iGPUs.
As tested by @Zoiudolo. (237 -> 300-320/s) Performance increase may differ by system.
